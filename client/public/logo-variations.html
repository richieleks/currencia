Logo Guidelines

  • Primary Color: #1565C0 (Blue 700)
  • Secondary Color: #0D47A1 (Blue 900)
  • Accent Color: #1976D2 (Blue 600)
  • Typography: Arial, sans-serif
  • Clear Space: Minimum 1/2 logo height on all sides
  • Minimum Size: 24px height for full logo, 16px for icon

Brand Concept

The Currencia logo represents:

  • Exchange: Curved arrows showing currency flow and conversion
  • Global: Multiple currency symbols ($, €, £, ¥) representing international reach
  • Professional: Clean, modern design suitable for B2B trading
  • Trust: Stable blue color palette conveying reliability and security
  • Technology: Modern SVG design scalable for all digital applications
